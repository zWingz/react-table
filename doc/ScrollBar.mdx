# ScrollBar

** If not effect, shrink the window to make it take effect **

## Description

- Show a virtual scroll bar when `scrollTarget` scrollbar hide
- Hidden when `scrollTarget` scrollbar show in window

## Base Usage, dependent on Window

<Playground>
  {
    () => {
      return <HorizontalScrollBar>
        <div style={{fontSize: '0px'}}>
          <img src={Image}></img>
        </div>
      </HorizontalScrollBar>
    }
  }
</Playground>

## Dependent on a scrollTarget

<Playground>
  {
    () => {
      return <div id='test-scroller' style={{overflow: 'auto', width: '600px', height: '600px'}}>
        <HorizontalScrollBar scrollTarget='test-scroller'>
          <div style={{fontSize: '0px'}}>
            <img src={Image}></img>
          </div>
        </HorizontalScrollBar>
      </div>
    }
  }
</Playground>

## Props

| Props | Type | Default  | Desc |
| ----- | ---- | ---- | ------- |
| className | string | '' | scrollbar className |
| scrollTarget | string &#124; HTMLElement | window | native scroll container |
| offsetBottom | number | 5 | scrollbar bottom offset |
